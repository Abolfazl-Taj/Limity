
# Limity Project

A spa multipurpose buisiness project created with ReactJs 


## Run Locally

Clone the project

```bash
  git clone https://github.com/Abolfazl-Taj/Limity.git
```

Go to the project directory

```bash
  cd Limity
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Features

- Light/dark mode toggle
- Fullscreen mode
- Cross platform
- Responsive
- Spa
- Using React Router Dom
- Nested Route 
- Animated
